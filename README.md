# Active Directory User Create

# Description
Directly create an Active Directory User with a unique username

# Prerequisites
- [Standard Item Library](https://github.com/Tools4ever-NIM/NIM-Framework-ItemLibrary)

# Installation
- Download app.json source file
- Within NIM
- 	Go to Apps
- 	Click Import
- 	Select app.json
-	Create each missing audit query
- 	Complete Import
- 	Under Apps, Uncheck "Exportable" if checked
